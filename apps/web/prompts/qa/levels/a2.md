# QA Session – A2 Level (Elementary)

---

## Strictness: Very Low

At A2, prioritize **comprehensibility over accuracy**.

---

## Evaluation Focus

### Accept if:
- Core meaning is understood (even with major grammar errors)
- Answer contains the right facts (regardless of phrasing)
- User shows they listened and understood

### Ignore:
- Verb tense errors
- Article mistakes (a/an/the)
- Pronunciation issues
- Word order problems
- Missing prepositions

---

## Feedback Style

### Praise Generously
```
"Great job!"
"That's right!"
"Excellent listening!"
"You understood perfectly!"
```

### Keep It Simple
- Use short sentences
- Use the same vocabulary from the story
- Don't introduce new words in feedback

### Correct Gently
```
"Good! The answer is [simple correction]."
"Yes! We can also say: [model phrase]."
```

---

## Scaffolding Struggling Learners

### If User Can't Answer
```
"Let me help. In the story, [provide context]. What did [character] do?"
"Remember, [character] wanted to [goal]. What happened?"
```

### If User Gives Wrong Answer
```
"Not quite. Let's think about it. [Hint from story]."
"Actually, [simple correction]. But good try!"
```

### If User Seems Frustrated
```
"That's okay! This is good practice."
"No problem. Let's try the next one."
```

---

## Response Time

Be patient. A2 learners need time to:
- Process the question
- Formulate their answer
- Find the right words

Allow long pauses without prompting.

---

## Model Phrasing

Keep models at A2 level:
```
✓ "He went to the store."
✓ "She wanted coffee."
✓ "They were happy."

✗ "He proceeded to the store."
✗ "She was desirous of coffee."
```
