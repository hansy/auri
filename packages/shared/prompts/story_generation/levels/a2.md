# Story Generation – A2 Level (Elementary)

---

## Learner Profile

A2 learners can:
- Understand simple sentences about immediate needs
- Communicate in routine tasks
- Describe their background and immediate environment

**They cannot:**
- Follow extended speech
- Handle rapid speech
- Infer meaning from unknown vocabulary

---

## Numeric Rubric

| Parameter | A2 Value |
|-----------|----------|
| Story length | 90–120 seconds (2 min) |
| Cast | 1 (monologue only) |
| Entity count | 1–3 |
| Time shifts | 0 |
| Inference load | minimal |
| Clause depth | simple |
| Lexical rarity | high_frequency |
| Discourse markers | basic |
| Audio speed | 90–110 wpm |
| Sentence pauses | 900–1200 ms |
| Chunk duration | 6–10 seconds |
| Discourse shape | linear |

---

## Questions

- **Count**: 2–3
- **Distribution**: All literal, at most 1 simple relational

### Patterns
```
✓ "Where did [character] go?"
✓ "What did [character] buy?"
✓ "When did this happen?"
✓ "Why did [character] go there?" (explicit reason only)
```

---

## Story Crafting

### Language
- High-frequency vocabulary only (A1-A2 word lists)
- Concrete nouns, no abstract concepts
- Present simple, past simple, "going to" future
- Avoid phrasal verbs except: get up, come back, go out
- No idioms
- One idea per sentence

### Structure
- Linear narrative only (chronological)
- Clear beginning, middle, end
- One main character, one main action
- Explicit cause-and-effect: "It was raining, so she took an umbrella"
- No flashbacks, no hypotheticals

### Discourse Markers
- Time: first, then, after that, finally
- Cause: so, because
- Addition: and, also

### Audio
- Slow and clear
- Long pauses (900-1200ms)
- Exaggerated intonation on key words
- Pronounce each word distinctly

---

## Scenario Adaptations

| Scenario | A2 Version |
|----------|------------|
| Scheduling | "I have a meeting on Tuesday at 3pm" |
| Making plans | "Let's go to the park on Saturday" |
| Service issues | "My order is late. Can you help?" |
| Clarification | "Sorry, can you repeat that?" |
| Choosing options | "I want the red one" |

---

## Red Flags (Never Use)

- ❌ Subordinate clauses ("The book that I bought...")
- ❌ Passive voice
- ❌ Reported speech
- ❌ Conditional sentences
- ❌ Abstract nouns
- ❌ Multiple speakers
- ❌ Implied meaning
