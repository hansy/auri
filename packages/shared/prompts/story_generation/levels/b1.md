# Story Generation – B1 Level (Threshold)

---

## Learner Profile

B1 learners can:
- Understand main points of clear standard speech on familiar matters
- Handle most travel situations
- Produce connected text on familiar topics
- Describe experiences and express opinions simply

**They are developing:**
- Following extended speech (need clear structure)
- Expressing opinions with justifications

---

## Numeric Rubric

| Parameter | B1 Value |
|-----------|----------|
| Story length | 150–180 seconds (3 min) |
| Cast | 1–2 (prefer monologue) |
| Entity count | 2–4 |
| Time shifts | 0–1 (clearly signposted) |
| Inference load | low |
| Clause depth | simple → moderate |
| Lexical rarity | common |
| Discourse markers | basic → natural |
| Audio speed | 110–140 wpm |
| Sentence pauses | 600–900 ms |
| Chunk duration | 8–12 seconds |
| Discourse shape | linear or problem_resolution |

---

## Questions

- **Count**: 3–4
- **Distribution**: Mostly literal + relational, rare pragmatic

### Patterns
```
✓ "What problem occurred?"
✓ "What decision did they make?"
✓ "Why did they change the plan?"
✓ "What would you do in this situation?"
✓ "What happened first/next/last?"
```

---

## Story Crafting

### Language
- Common vocabulary (beyond A2, avoid rare words)
- Some phrasal verbs: look for, find out, work out
- Present, past, future, present perfect
- Occasional passive when natural
- Some abstract nouns if clearly contextualized

### Structure
- Linear or problem-resolution narrative
- Clear problem statement early
- Explicit connection between problem and solution
- Time shifts must be signposted: "Last week...", "Now..."

### Discourse Markers
- Contrast: but, however, although
- Sequence: after, before, while, when
- Cause/Result: because, so, that's why
- Condition: if (simple first conditional)

### Dialogue (When Using 2 Speakers)
- Clear turn-taking, no overlap
- Distinct speaker roles
- Maximum 4-5 turns per speaker

---

## Scenario Adaptations

| Scenario | B1 Version |
|----------|------------|
| Scheduling | Rescheduling due to conflict, negotiating new time |
| Service issues | Explaining problem, requesting specific solution |
| Choosing options | Weighing 2 options with clear pros/cons |
| Giving advice | Advice with one caveat |

---

## Red Flags (Never Use)

- ❌ Complex conditionals ("If I had known...")
- ❌ Dense subordination (multiple embedded clauses)
- ❌ Implied criticism or sarcasm
- ❌ Rapid dialogue with interruptions
- ❌ Unstated conclusions
